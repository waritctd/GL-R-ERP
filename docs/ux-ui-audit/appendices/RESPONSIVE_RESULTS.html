Specific responsive issues

  • UX-01 — ticket detail leaves ~1,300px blank below content at 390px.
  • UX-05 — CeoSettings tables shrink instead of reflowing; Override column clipped.
  • UX-22 — 5-phase tracker truncates labels mid-word on mobile.
  • UX-10 — payroll/attendance render a DesktopOnlyNotice (no mobile UI).

What reflows well

DataTable mobileCard (attendance, tickets), the nav drawer, and HR self-service forms all adapt correctly; the Button component enforces a 44px touch floor.
